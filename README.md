# TikTokMall

backend: 后端
启动：
```shell
cd backend
make start
```


frontend: 前端

node 版本 v14.17.5

```shell
cd frontend
npm install
npm run serve
```



## 一、项目概述
本项目旨在构建一个功能完善的电商购物平台，涵盖用户认证、用户管理、商品管理、购物车、订单处理、支付结算以及 AI 大模型辅助等功能模块，以提供流畅的购物体验和智能化的服务。
